# Cyber Forensics Investigation Lab

This repository documents hands-on digital forensic investigations across multiple domains, including OSINT, disk forensics, network analysis, and data hiding techniques.

---

## Objective

To develop practical forensic investigation skills by analyzing evidence, reconstructing incidents, and documenting findings through structured case studies.

---

## Focus Areas

- OSINT (Open Source Intelligence) investigations  
- Disk and file system forensics  
- Network traffic analysis  
- Steganography and cryptography  

---

## Repository Structure

- cases/ – Investigation-based case studies  
- labs/ – Environment setup and datasets  
- notes/ – Concepts and reference material  
- tools/ – Tools and usage documentation  
- journal/ – Progress and observations  

---

## Approach

- Analyze real-world inspired scenarios  
- Use forensic tools to extract and interpret evidence  
- Document investigation steps and findings  
- Build structured and reproducible case reports  

---

## Goal

To build a portfolio of investigation-driven forensic work demonstrating practical cybersecurity and analytical skills.
