# Disk Forensics Cases

This section contains investigations involving disk images and file systems.

## Purpose

- Recover and analyze files  
- Identify artifacts and hidden data  
- Reconstruct user activity  

---

## Focus

- Disk image analysis  
- File recovery (allocated and unallocated space)  
- Metadata examination  
- Evidence extraction  

---

## Goal

To build practical skills in analyzing storage media and extracting digital evidence.
