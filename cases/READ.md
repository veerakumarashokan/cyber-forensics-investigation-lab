# Cases

This section contains structured forensic investigations across different domains.

## Purpose

- Document investigation workflows  
- Analyze evidence and artifacts  
- Present findings in a structured format  

---

## Categories

- osint/ – Open source intelligence investigations  
- disk-forensics/ – Disk and file system analysis  
- network-forensics/ – Network traffic analysis  
- steganography/ – Hidden data and cryptographic analysis  

---

## Approach

Each case includes:
- scenario description  
- tools used  
- investigation steps  
- findings and conclusions  

---

## Note

All cases are based on simulated or assignment-driven scenarios designed to reflect real-world investigations.
