# OSINT Cases

This section contains investigations based on publicly available information.

## Purpose

- Analyze websites, domains, and online presence  
- Identify potential risks and red flags  
- Understand attacker or target behavior  

---

## Focus

- WHOIS analysis  
- Website history (Wayback Machine)  
- Search-based intelligence (Google dorking)  
- Network mapping (Maltego)  
- Social media investigation  

---

## Goal

To develop structured OSINT investigation skills using publicly available data.
