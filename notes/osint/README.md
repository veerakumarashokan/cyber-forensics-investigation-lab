# OSINT Notes

This section contains concepts and reference material related to open source intelligence (OSINT).

## Purpose

- Understand information gathering techniques  
- Analyze publicly available data  
- Support OSINT-based investigations  

---

## Focus

- WHOIS and domain analysis  
- Website history (Wayback Machine)  
- Search-based intelligence (Google dorking)  
- Social media analysis  
- Network mapping  

---

## Goal

To build structured knowledge for performing effective OSINT investigations.
